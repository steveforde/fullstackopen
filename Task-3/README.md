# Full Stack Open - Phonebook-Stephen Forde

This repository contains the backend and frontend for the Phonebook application.

## Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Database:** MongoDB (Atlas)
- **Deployment:** Render
- **Tooling:** ESLint, Git

## Live Application

The application is deployed to Render: https://phonebook-backend-step9.onrender.com

Note: \***\*\*\*\* The backend is hosted on Render's free tier. Please allow up to 1-2 minutes for the server to "wake up" on the first request. **\*\*\*\*\*
